## 🚀 亮点概述
这份配置基于 **Clash Meta** 构建，围绕「**稳定、干净、智能、精细分流**」四个核心目标设计。

它相比普通 Clash 配置拥有更高的自动化程度、更干净的节点列表、更细的分流策略以及更好的 DNS 与系统兼容性。

---

## ⭐ 核心优势一览

### 🧠 1. **高度自动化**
- 多订阅源合并管理  
- 健康检查自动剔除失效节点  
- 正则过滤排除广告、返利、促销等垃圾节点  
- 地区节点自动测速（url-test 动态优选最优节点）  

**→ 节点永远保持干净、可用、自动挑选最优。**

---

### 🧩 2. **分流体系精细到服务级别**
常见应用全部独立分组：

- YouTube / Google / Twitter / Telegram / TikTok  
- Netflix / Spotify / Instagram / Github  
- ChatGPT & OpenAI 单独策略组  

**→ 不同服务走最适合地区的节点，体验更顺滑。**

---

### 🌍 3. **智能地区节点池（自动测速）**
HK / TW / JP / SG / US 全部采用：

- `url-test` 自动测速  
- 定时更新最佳线路  
- filter 匹配地区关键字  
- 完整图标美化（UI 友好）  

**→ 无需切换节点，系统自动帮你选最快。**

---

### 🧬 4. **DNS 高级优化：Fake-IP + 国内外分流**
- Fake-IP 提升跨境访问效率  
- 国内域名走国内 DNS（阿里/腾讯）  
- 国外域名走 Cloudflare / Google  
- 多重 fallback 提升稳定性  

**→ 国内服务不卡顿，国外网站更稳定。**

---

### 🖥️ 5. **TUN 模式接管全局流量（适合 Windows / Mac）**
- 系统全流量自动代理  
- 自动路由  
- 自动 hijack DNS  

**→ 本地软件无需单独设置，全局透明代理体验。**

---

### 🧱 6. **结构清晰、注释完整、方便二次开发**
相较于网上的模板，本配置更适合长期维护：

- 逻辑严谨  
- 注释详细  
- 易理解、易调整  

**→ 适合从新手到进阶用户长期使用。**

## 🔍 模块结构说明

### 📦 1. Proxy Providers（订阅管理）
- 合并多个订阅源  
- 健康检查  
- 正则过滤  
- 自动前缀清洗  

### 🔗 2. 自定义节点
内置自有 SS / IEPL 节点，可作为稳定保底线路。

### ⚙️ 3. 基础配置
- IPv6 支持  
- Chrome 指纹  
- Web UI 控制台可直接访问  

### 🕵️ 4. Sniffer 流量识别
识别 TLS/QUIC/HTTP，提高分流命中率。

### 🌀 5. TUN 系统代理
全设备透明代理，兼容性最强。

### 🌐 6. DNS 分流系统
国内外分开解析，提高速度与稳定性。

### 🧭 7. Proxy Groups（策略组）
- 总控组（Proxy / Final）  
- 应用分流组  
- 地区节点池（自动测速）  

### 📜 8. 完整规则系统
包含：
- 国内直连
- 全球加速
- 特定应用规则
- 最终兜底规则
